# Platinum Standard - Flywheel

## Loop

```text
Capture (RDTI, decisions, pipeline outcomes)
 -> Curate (company memory, tags, confidence)
 -> Improve prompts/skills from real failures
 -> Evaluate (weekly board metrics)
 -> Hot-swap skill versions
```

| Source | Writer skill |
|--------|--------------|
| Incorporation milestones | formation-officer |
| Deadlines | compliance-registrar |
| R&D hours / uncertainty | grants-rdti-clerk |
| Market evidence | market-validator |
| Pipeline stage changes | gtm-pipeline-rep |
| Content performance notes | content-comms-officer |
| Runway / burn | finance-clerk |
| Investor interactions | funding-analyst |
| Contract drafts registry | legal-document-assistant |
| Weekly synthesis | board-chief-of-staff |

Without capture, Platinum is marketing. With capture, the fleet compounds.
