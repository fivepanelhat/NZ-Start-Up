# Gold Standard — Workflow Execution

## Phase gates (linear)

1. **Discovery** — problem, ICP, constraints  
2. **Design** — map real-world founder steps to skills/templates  
3. **Development** — implement skill/template changes  
4. **Testing** — validate_skills + pytest + manual HITL spot-check  
5. **Deployment** — version bump, changelog, release  

Cannot claim phase N+1 complete without phase N evidence.
