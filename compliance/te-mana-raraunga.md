# Te Mana Raraunga Overlay

Applies to all CAT tiers for this product.

## Principles → product rules

| Principle | Rule |
|-----------|------|
| Rangatiratanga | Founder and Māori partners control their data and framing |
| Whakapapa | Memory retains relational context; no decontextualised “Māori data” dumps |
| Whanaungatanga | Iwi/hapū engagement is relationship-first, not agent spam |
| Kotahitanga | Cohort/white-label serves capacity building |
| Manaakitanga | Careful language; no extractive branding |
| Kaitiakitanga | Prefer local processing; document any offshore flow |

## Skill behaviour

1. Do not invent iwi endorsement or Māori partnership.  
2. If user claims Māori pathway, set `cultural_review_required: true` on drafts.  
3. Reject designs that silently export whenua-linked or Māori data offshore.  
4. HITL for cultural content publication.
