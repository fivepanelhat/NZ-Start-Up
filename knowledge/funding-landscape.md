# NZ Funding Landscape (Starter Knowledge)

**Status:** Starter snapshot for agent context. Re-verify every opportunity before drafting. Post-Callaghan reshuffles mean names and portals change.

## Categories

| Category | Examples (illustrative) | Clerk mode |
|----------|-------------------------|------------|
| R&D tax incentive | RDTI (rdti.govt.nz) | Continuous logging |
| Regional EDA | ScaleUp-style grants, capability vouchers | Discover + draft |
| Accelerators | PowerUp, Sprout, Icehouse programmes | Cohort calendars |
| Sector | AGMARDT, agritech, deeptech funds | Fit-score |
| Capital | Angels, NZGCP pathways, SAFE pre-seed | funding-analyst |
| Māori development | Whenua / iwi-linked funds | Cultural review required |

## Fit scoring (0–100)

Dimensions (see grants skill references):

1. Eligibility hard gates (entity type, location, stage)  
2. Technical uncertainty / R&D character  
3. Co-fund rules (no double-dip lies)  
4. Sovereignty fit (data residency)  
5. Capacity to deliver milestones  
6. Strategic wedge alignment  

## Dogfood rule

Run RDTI logging for Coastal Alpine Tech (or the founder’s real R&D) from week one — retroactive logs fail claims.
