# NZ Startup Launch Sequence

Living operating system for the first 90 days post-incorporation. Turns the 30/60/90 plan into weekly executable priorities and a single source of truth for progress.

## Core Cadence

- **Weekly Board Pack** (generated every Sunday or Monday): pipeline vs plan, cash/runway, compliance deadlines, top-3 priorities for the coming week, blockers.
- **Daily focus rule**: protect time for the single highest-leverage activity (usually a discovery conversation, demo build, or grant application).
- **Day gates**:
  - Day 0 (8 Aug): Incorporate + same-week bank/IRD/GST/Xero/insurance
  - Day 30: Legitimacy + 8 discovery conversations booked/held, demo unit filmed, ScaleUp applied
  - Day 60: 2 signed pilot LOIs, first RDTI documentation filed, pilot hardware ordered
  - Day 90: Clear path to the two paid field pilots and first revenue

## When to Load

- Start of week planning
- Preparing a board-style update for self or advisors
- Checking whether current activity is still on the critical path to the $25k MRR stretch goal (or the $8–12k base case)

## Instructions

1. Read the current 30/60/90 plan and any existing task/memory state.
2. Produce a short weekly pack with:
   - Progress against the three gates
   - Pipeline health (named targets → conversations → LOIs)
   - Funding & legitimacy actions completed/outstanding
   - Recommended top-3 priorities for the next 7 days
3. Route any specialised work to the appropriate skill (formation, Venture Taranaki, partnership, grants, etc.).
4. End with a clear Human Action Checklist for the founder.

## Guardrails

- Never invent progress. Only report what is evidenced in memory or provided by the founder.
- Keep the stretch goal ($25k MRR) visible but always show the base-case path.
- Protect founder energy: flag if the plan is becoming too wide.

## Success Criteria

- Founder always knows the single most important thing to do this week.
- No silent drift away from the wedge (Weaver + AquaGuard) or the anchor-account strategy.
